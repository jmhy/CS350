This folder contains the project files for Lab 2 - Advanced Maze Stage 1 (Factory Method) for CS 350.
Import the project into Eclipse and the entry point will be contained in MazeGame.java.
In the command line, if no arguments are given, then a normal predefined maze is made.
If one argument is given, then it is expected to be a maze type: "basic", "red", or "blue". It also defaults to basic if something else it given.
If two are given, the first is type and the second is the file path to load a maze from.
Example: "red C:\Users\user\documents\large.maze"
