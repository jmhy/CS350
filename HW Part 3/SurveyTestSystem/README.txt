This folder contains the project files for Homework 1 Part 3 for CS 350.
Import the project into Eclipse and the entry point will be contained in Main.java.
Surveys are stored in the "surveys" folder, tests in the "tests" folder.
Files are saved and loaded using the Serialization interface.
1 sample survey and 1 sample test are provided.
Users' responses to each survey and test are stored in the survey/test object, thus are stored in the survey/test files.
Essays do not get graded, but the number of essays will be displayed.

No known issues.
